## 部署到 Vercel 步驟

1. 去 https://github.com 註冊/登入
2. 點右上角 + → New repository
3. 名稱填 `couple-finance`，按 Create repository
4. 把這個資料夾裡的所有檔案上傳（點 uploading an existing file）
5. 去 https://vercel.com，用 GitHub 帳號登入
6. 點 Add New → Project → 選 couple-finance
7. 直接按 Deploy
8. 完成！會拿到一個網址如 https://couple-finance-xxx.vercel.app
